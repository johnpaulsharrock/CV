Project Synopsis

Software Development

Design a performance management system, then evaluate its development within a MEAN stack

This project saw a solution being designed using traditional OOP techniques which were captured within a Volere template. The intent was then to develop the proposed solution within a MEAN stack technology. A prototype was developed using an iterative development cycle and deployed after the first cycle. The client for this project was interested in adopting MEAN stack technologies and therefore an evaluation of the technology was conducted using a set criteria. In addition, the feasibility of using classical OOP design techniques more suited to opinionated technologies -such as Java, was evaluated in its use for un-opinionated technologies, in this instance Node.js and Express.
